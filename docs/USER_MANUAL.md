# User Manual

## Management Users

Open `/management` to view the operations dashboard.

Use the dashboard modules to manage:

- Maintenance requests
- Work orders
- Assets
- Inventory
- Scheduling
- EQP preventive maintenance reports

## Field Technicians

Open `/technician/login`.

The technician interface is Arabic-first and right-to-left:

- عرض المهام
- بدء المهمة
- تعبئة قائمة الفحص
- إضافة ملاحظات عربية
- إغلاق المهمة

Arabic labels are translated into English internal statuses for reporting and audit consistency.

## EQP Users

Open `/eqp`.

Existing EQP behavior is preserved:

- Generate Excel reports
- View report archive
- Rename reports
- Delete reports
- Review machine history
