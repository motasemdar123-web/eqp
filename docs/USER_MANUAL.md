# User Manual

## Management Users

Open `/` and choose `Continue with Microsoft`. Use your Dar Al HAI Microsoft account.

Open `/management` to view the operations dashboard.

Use the dashboard modules to manage:

- Maintenance requests
- Work orders
- Assets
- Inventory
- Scheduling
- Engineer completion approvals
- EQP preventive maintenance reports

## Engineers

Open `/engineer`.

Use the page to:

- review jobs submitted by technicians for completion
- inspect technician Arabic notes
- open completion photos
- approve completed work
- return work to the technician with review notes

## Scheduling

Open `/management/scheduling`.

Use the page to:

- sign in with Microsoft when needed
- view the daily technician roster
- create shifts
- add or update technician daily schedules
- create job cards
- assign one or more technicians
- define work hours, task scope, safety notes, required tools, required parts, and permit requirements

The seeded demo environment includes 9 technicians and a daily schedule for each technician.

## Technicians

Open `/technician`.

The technician page is Arabic-first and right-to-left. Technicians can:

- view their daily schedule
- view assigned job cards
- add Arabic completion notes
- upload completion photos
- submit the job to the responsible engineer for approval

Submitted jobs move to `PENDING_APPROVAL` until an engineer approves or returns them.

## EQP Users

Open `/eqp`.

Existing EQP behavior is preserved:

- Generate Excel reports
- View report archive
- Rename reports
- Delete reports
- Review machine history
